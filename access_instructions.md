# Accessing the Credence Exchange Dashboard

If `http://localhost:8085` did not load, please try one of the following methods. 
This is usually caused by local hostname resolving differences on Windows (IPv6).

---

### 🌐 Method 1: Use direct Loopback IP (Recommended)
Try clicking or pasting these direct IP links instead of `localhost`:
- **[http://127.0.0.1:8085](http://127.0.0.1:8085)**
- **[http://127.0.0.1:8086](http://127.0.0.1:8086)** (Alternative port)

---

### 📂 Method 2: Open File Directly (No Server)
You can view the dashboard without any network server simply by opening the static HTML file on your drive:

1. Open your File Explorer
2. Navigate to: `c:\Users\mihsa\OneDrive\Documents\Google AG\Journalism in AI age\design\04_artifact_04_style4\`
3. **Double-click** `index.html` to open it instantly.
